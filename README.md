**项目核心框架介绍 : python + playwright + PO + pytest + allure + jenkins**

1. 提供基础框架封装（面向对象）：base层，主要页面基础操作方法封装 

   - 封装公共方法：打开网页、输入操作、点击操作、获取新窗口、截图等

2. 提供页面对象封装（PO）：pages层，页面对象封装，主要封装页面操作方法

   - 封装每个页面对象：
     - 元素定位（属性）
     - 页面操作（方法）【业务】

3. 提供页面脚本层（pytest）：scripts层，主要封装测试的页面业务脚本

   - 封装测试脚本：测试用例（准备数据、调用测试方法、打印日志、断言）

4. 其他层：

   - data：测试数据层 

   - report：测试报告层

   - log：日志层

   - img：截图层

   - config.py：是项目配置信息（存放项目相关的公共配置信息）

   - pytest.ini：是pytest配置文件

   - conftest.py：全局管理测试用例的前后置操作

5. 公共工具层（utils）：tools操作日志记录、读取JSON文件
